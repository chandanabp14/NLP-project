News Analyzer using NLTK and spaCy

Features:
- Tokenization
- Stop-word Removal
- Word Frequency Analysis
- POS Tagging
- Named Entity Recognition
- Sentiment Analysis

Libraries Used:

NLTK:
- Tokenization
- Stop-word Removal
- Frequency Analysis

spaCy:
- POS Tagging
- Named Entity Recognition

TextBlob:
- Sentiment Analysis

Installation

1. Install dependencies
pip install -r requirements.txt

2. Download spaCy model
python -m spacy download en_core_web_sm


How to run
python app.py
